AllRoutes Logistics

A live, production-grade cross-platform logistics platform I designed, built, and launched entirely alone — now serving an active rider network across Lagos with real reviews from real customers.

AR
AllRoutes Logistics
Last-mile delivery platform · Lagos, Nigeria
🌐 Live Website ▶ Google Play ⬛ App Store

AllRoutes is a complete logistics ecosystem — not a side project. It handles real orders, real payments, real riders, and real customers every day. I built every layer of this platform independently over the course of one year.

Cross-platform Flutter app — live on Google Play Store and iOS App Store with real-time order management and live GPS tracking
Node.js/TypeScript REST API — scalable backend handling rider authentication, order lifecycle, payment processing, and Socket.io real-time communication
React/TypeScript admin dashboard — customer and operator-facing web platform with full order visibility
AWS Rekognition + S3 face verification — ML-powered rider identity confirmation during onboarding using live selfie matching
Google Maps API — route polylines, geolocation, and live rider tracking for customers
Firebase Cloud Messaging — push notifications for order updates, rider assignments, and delivery confirmations
Payment gateway integration — rider wallet management, debt tracking, and same-day payment processing
Flutter Node.js TypeScript React AWS Socket.io Firebase Google Maps MongoDB
